## 什么是 Favicon 下载器？

一个简洁、快速的在线工具，帮助你立即提取并下载任何网站的 favicon 图标。支持多种尺寸，从 16x16 到 512x512，满足各类项目需求。

## 核心功能

- **即时提取**：输入域名，一键获取所有可用的 favicon 尺寸
- **多尺寸支持**：涵盖 16x16、32x32、192x192、512x512 等常用规格
- **一键下载**：选择任意尺寸，直接下载到本地
- **代码生成**：自动生成标准 HTML 引用代码
- **完全免费**：无需注册，随时使用

## 使用方法

1. 在输入框输入目标域名（如 example.com）
2. 点击「获取 Favicon」按钮
3. 查看所有可用尺寸的图标预览
4. 点击「下载」保存所需尺寸
5. 复制 HTML 代码，快速集成到项目中

## 为什么选择我们？

- **节省时间**：无需手动查找源代码或服务器目录
- **结果全面**：自动检测并展示所有可用图标尺寸
- **开发友好**：提供即用 HTML 代码片段
- **设计参考**：快速获取竞品或优秀网站的图标设计

## 可用尺寸说明

本工具会动态扫描目标网站的 favicon 实现，展示所有检测到的尺寸。常见规格包括：

- 16×16（经典桌面浏览器标签页）
- 32×32（高分辨率标签页）
- 180×180（iOS Safari 快捷方式）
- 192×192（Android Chrome 桌面图标）
- 512×512（PWA 应用图标）

实际可用尺寸因网站而异，工具会完整展示检测结果。
