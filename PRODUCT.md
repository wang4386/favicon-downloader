# Product

## What

Favicon Downloader API — 一个简洁的工具，输入域名，获取并下载该网站的 favicon 图标（多种尺寸）。

## Who

开发者、设计师、产品经理，需要快速获取网站图标用于文档、设计稿、书签、应用集成等场景。

## Register

**Product** — 这是一个工具型应用，设计服务于功能。用户来这里是为了完成任务（获取图标），不是为了欣赏界面。

## Core flows

1. **输入域名** → 输入框，一键获取
2. **查看结果** → 显示默认尺寸和较大尺寸的图标预览
3. **API 调用说明** → 每个尺寸卡片展示对应的 API 调用代码

## Constraints

- 默认语言：中文（无需语言切换）
- 无需明暗主题切换（单一主题）
- 移动端优先，响应式设计
- 核心功能：输入 → 获取 → 展示（三步完成）

## Brand notes

- 工具型界面，干净、专业、高效
- 不追求花哨，但要有精致感
- 代码示例清晰易读，一眼看懂
